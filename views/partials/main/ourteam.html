Backend Project

The project is an e-commerce website built with JavaScript, Node.js, Express.js, Stripe, Express-ES6-Template-Engine, Bcrypt, Cookie-Parser, and Express Session. Both users and admins can create accounts, and users can securely purchase items using Stripe. The website uses Bcrypt for password security, Cookie-Parser and Express Session for session management.
